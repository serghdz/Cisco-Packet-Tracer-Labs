# Cisco-Packet-Tracer-Labs
Documenting my networking learning journey using Cisco Packet Tracer

A collection of **Cisco Packet Tracer** labs to learn and practice networking fundamentals.

This repo documents hands-on labs using **Cisco Packet Tracer** to build, configure, and troubleshoot network topologies. Perfect for beginners learning switching, routing, VLANs, and more.

---

## Lab Topics
- VLANs & Trunking
- Inter-VLAN Routing
- Static & Dynamic Routing (RIP, OSPF)
- DHCP, NAT, and ACLs
- Basic Network Services
